# Magical Forest Adventure

ブラウザだけで動く、オリジナルの横スクロール魔法アクションゲームです。

## 遊び方

1. `index.html` をブラウザで開きます。
2. 操作します。

## 操作

- ← → / A D：移動
- Space / W / ↑：ジャンプ
- J / Z：魔法弾
- R：リスタート

## 内容

- 魔法使い風のオリジナル主人公
- 横スクロールステージ
- 足場、穴、水トラップ、敵、星アイテム
- ジャンプ踏みつけ / 魔法弾で敵を倒す
- ゴール扉に触れるとクリア

## 注意

このゲームはオリジナル作品です。特定の既存キャラクター、作品名、ロゴ、音楽、画像素材は使用していません。
